# Home Page Proportion Pass

Date: 2026-05-15
Screen: home-page

## Goal

Improve the proportions of the homepage so the three main navigation cards feel wider, lighter, and less boxy.

## Problems Found

1. The cards were too close to square proportions.
2. The page width and grid width did not give the cards enough horizontal room.
3. The card CTA pills were slightly undersized compared to the mass of the cards.
4. The result felt heavy and a bit awkward instead of ceremonial and confident.

## Fixes Implemented

### 1. Wider homepage shell

Expanded the homepage width so the feature cards have more room to breathe on desktop.

### 2. Wider card grid

Kept the three-column structure, but widened the overall grid and increased gap rhythm to reduce the cramped feeling.

### 3. Lighter card proportions

Adjusted card padding and spacing so they read more like wide navigation sections than square boxes.

### 4. Stronger CTA proportions

Made the card buttons a bit taller and broader so they feel anchored to the card composition.

### 5. Mobile preserved

Added mobile-specific overrides so the wider desktop treatment does not break the phone layout.

## Result

The homepage should now feel less blocky, more panoramic, and more aligned with the ceremonial visual language of the rest of the updated site.
