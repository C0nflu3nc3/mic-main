# Studios Page Design Pass

Date: 2026-05-15
Screen: studios-page

## Goal

Bring `Студии` into the same visual family as `История и кодекс` and the newly harmonized ceremonial pages.

## Problems Found

1. `Студии` still used a different heading font treatment than the rest of the updated site.
2. The hero felt related to the older page system instead of the ceremonial one.
3. The empty state card looked technical and unfinished instead of editorial and intentional.

## Fixes Implemented

### 1. Unified page shell

Connected `Студии` to the shared `ceremonial-page` system so it inherits:

- stronger hero composition
- richer card atmosphere
- consistent spacing
- same visual hierarchy as other upgraded pages

### 2. Fixed heading font

Changed `Студии` headings back to the display heading font used across the rest of the redesigned sections.

This removes the mismatch where `Студии` looked softer/older than pages like `Бонусная система`.

### 3. Improved empty state

The “no studios yet” block is now:

- centered
- narrower
- more deliberate
- visually closer to an editorial panel than a default placeholder

## Result

`Студии` should now feel like part of the same imperial design system instead of a page with its own typography and fallback layout behavior.

## Next Good Pass

If we want to push it further:

1. Add portrait/image framing specifically for studio cards.
2. Create a more gallery-like studio layout on desktop.
3. Add richer hover states so studio cards feel more premium.
